# 🎓 SkillForge Academy

A full-stack Learning Management System (LMS) built using the MERN Stack. SkillForge Academy allows students to register for courses, manage their learning journey, and enables administrators to manage course registrations efficiently.

---

## 🚀 Features

### 👨‍🎓 Customer
- User Registration & Login
- Browse Available Courses
- Course Registration
- View Cart
- My Courses
- Course Details
- Profile Dashboard
- Notifications
- Reviews & Ratings
- Logout

### 👨‍💼 Admin
- Secure Admin Login
- Dashboard
- Manage Student Registrations
- Accept / Reject Course Requests
- Revoke Course Access
- CRUD Operations
- View All Users

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/harsha-guntreddi/skillforge-academy.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
